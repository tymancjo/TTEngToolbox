F3D Help

Text Format (Import / Export)

Name; color; current; width; height; x,y,z; dx,dy,dz; ...

  • Name — unique conductor identifier
  • Color — red, green, blue, yellow, orange, purple, cyan, magenta, white, gray
  • Current — Amperes; negative reverses direction
  • Width / Height — cross-section in mm
  • Waypoints — first point absolute; subsequent points are relative deltas

Phase Grouping

Add ph Name before a group of conductors to assign them to that phase. Phase totals appear in Results.

3D Interaction

  • Ortho / Persp button toggles orthographic (parallel) vs perspective projection — viewpoint is preserved
  • Click a conductor row or result entry to select it
  • Click a gold waypoint sphere in the viewport — X (red), Y (green), Z (blue) axis arrows appear
  • Drag an axis arrow to move the waypoint along that axis — snaps to 5 mm grid
  • Drag the first (origin) waypoint to translate the whole conductor without reshaping it
  • While a waypoint is active, click the blue + button (top-center of viewport) to append a new waypoint at the end; it becomes selected immediately so you can drag it into place
  • Edit coordinates precisely in the Waypoints table, then click Apply
  • ⧉ button in Properties duplicates the selected conductor (offset +20 mm on X)
